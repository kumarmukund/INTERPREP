--> Compiler translates one representation of program to another representation. From High level language to low level machine code object. They are just the part of program development environment.
-->Source code is normally optimized for humans. Expressive : Match our notation of language. Redundant : Helps to avoid programming errors.
--> Machine code is optimized for hardware. Redundancy is reduced & Information about the intent is lost.

Having good compile time performance. Having maintainable code and High level of abstraction. Conversion is done on small steps.
