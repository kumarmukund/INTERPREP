WHAT IS MACHINE LEARNING ?

It is the science of getting computers to learn and acts like humans do, & improve their learning overtime in autonomous fashion, by feeding them data & information in form of observation and real world intraction.
It is an automated process & computer tries to learn everything without being explicitely programmed.

WHAT IS TRADITIONAL PROGRAMMING ?

In this user gives data and program to machine for getting it's output.

HOW MACHINE LEARNING WORKS?

In this an input data is feeded under an algorithm. An algorithmis set of rules which tells us about how to set data in program. An algorithm finds out the pattern inside the data & classify them.
In testing phase, machine is introduced with new data set which can't be seen by machine learning model till now. Machine learning model will imply it's all learning & generate output.

SUPERVISEED LEARNING

In Supervised learning at every instance we can say what is the output.
It is of two types :-

  • Classification :- when the output variable is categorized with two or more classes, e.g yes/no
  • Regression :- Relationship between two or more variable, where a change in one variable is associated with another variable.
Label data :- The data for which you already know the answers.

APPLICATIONS SUPERVISEED LEARNING
  •  Risk assesment.
  •  Image classification.
  •  Fraud detection.
  •  Visual recognition.
UNSUPERVISEED LEARNING

In Unupervised learning , an algorithm is trained using data that is unlabel. Machine tries to identifies the pattern.
They are of two types :-

  •   Clustering :- The method of dividing the objects into the clusters which are similar betweeen them and are dissimilar to the oobjects belonging to one another.
  •   Association :- Discovering the probability of co-occurance of items in a collection.

APPLICATIONS OF UNSUPERVISEED LEARNING
  •  Market based analysis.
  •  Semantic analysis
  •  Delivery store optimization.
  •  Identifying accident prone areas.
